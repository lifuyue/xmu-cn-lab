# xmu-cn-lab

计算机网络实验总目录。

## 文件导览

- `LAB1/`
  - 基于 OpenCV 的“屏幕发光 + 摄像头收光”颜色通信实验
  - 进入目录后可使用 `make` 和 `make test`
- `LAB2/`
  - RS232C 7-bit 电压序列编码与解码实验
  - 进入目录后可使用 `make` 和 `make test`
- `LAB3/`
  - `25/`：两路二值消息序列的多路复用与解复用实验
  - `26/`：正弦载波与模拟/数字调制实验
- `LAB4/`
  - 奇偶校验码检验实验
- `LAB5/`
  - `1/`：以太网帧目的 MAC 地址匹配判断
  - `2/`：传统以太网帧头部定义
  - `3/`：CSMA/CD 多线程模拟
- `LAB6/`
  - `1/`：网桥自学习与转发模拟
  - `2/`：教学楼 AP 部署与热力图生成
- `LAB7/`
  - `29/`：网桥自学习功能模拟

## 当前 LAB 列表

- `LAB1`：颜色编码光通信模拟
- `LAB2`：RS232C 7-bit 编码与解码
- `LAB3`：多路复用与调制实验
- `LAB4`：奇偶校验码检验
- `LAB5`：以太网帧与 CSMA/CD 实验
- `LAB6`：网桥学习与无线 AP 部署实验
- `LAB7`：网桥自学习功能模拟

后续新的实验可以继续按 `LAB2`、`LAB3` 这样的目录结构追加。
